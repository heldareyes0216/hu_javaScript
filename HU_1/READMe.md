# Programa de Validación de Edad en Node.js

Este es un pequeño programa en **Node.js** que interactúa con el usuario por consola, solicitando su nombre y edad, validando la entrada y mostrando un mensaje personalizado según si es mayor o menor de edad.

---

## ¿Qué hace este proyecto?

- Solicita el nombre del usuario por consola
- Solicita la edad del usuario
- Valida que la edad sea un número válido
- Usa un ciclo para evitar entradas incorrectas
- Muestra un mensaje personalizado:
  - ✔ Mayor de edad
  - ✔ Menor de edad

---

## Conceptos utilizados

- Node.js
- Módulo `readline`
- Funciones asincrónicas (`async / await`)
- Promesas (`Promise`)
- Validación de datos
- Condicionales (`if / else`)
- Ciclos (`while`)
- Entrada y salida por consola
