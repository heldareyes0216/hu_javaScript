# Admin de Productos - Dashboard CRUD

Aplicación web desarrollada en JavaScript puro (Vanilla JS) que simula un sistema de administración de productos tipo dashboard. Permite crear, listar y eliminar productos, con persistencia en LocalStorage y sincronización con una API simulada usando JSON Server.

---

## Funcionalidades

-  Agregar productos (nombre y precio)
-  Listar productos dinámicamente en el DOM
-  Eliminar productos individualmente
-  Dashboard con estadísticas:
  - Total de productos
  - Valor total del inventario
  - Promedio de precios
-  Persistencia con LocalStorage
-  Sincronización automática con API (JSON Server)
-  Consumo de API con Fetch (GET, POST, DELETE)
-  Manejo de errores con try/catch
-  Interfaz tipo admin panel (dashboard moderno)

---

##  Tecnologías usadas

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript ES6+
- Fetch API
- LocalStorage
- JSON Server
