# Manejo de Estructuras de Datos en JavaScript

Este proyecto es una práctica de JavaScript donde se trabajan diferentes estructuras de datos y métodos de iteración, incluyendo objetos, arrays, Sets y Maps, junto con validaciones de datos.

---

## ¿Qué hace este proyecto?

- Valida una lista de productos usando funciones
- Trabaja con estructuras de datos como:
  - Arrays
  - Objetos
  - Set
  - Map
- Elimina duplicados usando Set
- Relaciona categorías con productos usando Map
- Recorre estructuras con diferentes métodos (`for`, `for...of`, `for...in`, `forEach`)
- Usa métodos de objetos (`keys`, `values`, `entries`)
- Imprime resultados en consola
